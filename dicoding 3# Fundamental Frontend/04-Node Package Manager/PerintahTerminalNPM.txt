 ## Perintah NPM pada terminal untuk Project ##
- npm init
- npm install http-server --save-dev
- npm run start
- npm install http-server -g



** Hapus package
- npm uninstall http-server --save-dev  || npm uninstall -h --save-dev
- npm uninstall -h
- npm list -g
- npm list -g --depth=0

## Using and Installing Webpack ##
- npm install webpack --save-dev
- npm install webpack-cli --save-dev || npm install webpack webpack-cli --save-dev
- npm run build
- npm install style-loader css-loader --save-dev
- npm install @babel/core babel-loader @babel/preset-env --save-dev
- npm install html-webpack-plugin --save-dev
